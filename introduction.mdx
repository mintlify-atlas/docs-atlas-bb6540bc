# TradeMaster Transactions (TMT)

TradeMaster Transactions is an all-in-one SaaS platform designed for event organizers, venue operators, and ticketing companies. It combines event management, ticket sales, financial tracking, and user administration into a single, role-based system.

## What is TMT?

TMT is built for organizations that sell tickets to events — concerts, theater shows, sports games, conferences, and more. It provides a centralized back-office to manage:

- **Events** — Create, configure, and publish events with seating zones and pricing splits
- **Tickets** — Issue, validate, and search tickets with QR code support
- **Orders & Transactions** — Track every sale and financial movement in real time
- **Users** — Manage staff, clients, collaborators, and end customers
- **Offices (Taquillas)** — Run physical ticket offices with offline ticket support
- **Payouts** — Handle client payouts and custody accounts
- **Marketing** — Create and manage promotional campaigns
- **Portals** — Deploy branded client portals for end-user ticket access

## Platform architecture

TMT is a React-based single-page application backed by Firebase for real-time data storage and Auth0 for identity management. Role-based access control is enforced throughout using CASL, ensuring each user only sees and interacts with what they are permitted to.

## Key capabilities

- **Role-based access control** — Fine-grained permissions using CASL ensure each user type (Staff, Client, Collaborator, Customer) sees only what they are authorized to access.
- **Real-time data** — Firebase Firestore powers live dashboards and data updates across the platform.
- **Multi-language support** — The platform supports internationalization via i18next, making it suitable for multi-region deployments.
- **PDF generation** — Reports, contracts, and tickets can be exported as PDFs directly from the platform.
- **QR code ticketing** — Tickets include QR codes for fast, reliable entry validation at events.
- **Offline ticket support** — Physical offices (taquillas) can operate with offline ticket issuance when connectivity is limited.
